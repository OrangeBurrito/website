Installation Guide

  1. Download and install the Technic Launcher
  2. Launch TechnicLauncher and login to your Minecraft account
  3. Go to Launcher Options (top right corner) > Java Settings
  4. Set memory to 5GB
  5. Search for "RadRealms Server Pack" in the modpack search menu
  6. Install the modpack and press play (avg. loading time is about ten minutes)
  7. Join mc.radrealms.com and start playing!
